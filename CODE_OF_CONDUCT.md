# 贡献者行为准则

## 我们的承诺

为了培养一个开放和欢迎的环境，我们作为贡献者和维护者承诺，无论年龄、身体状况、残疾、种族、性别认同和表达、经验水平、国籍、外貌、种族、宗教，还是性取向如何，参与我们项目和社区的每个人都享受无骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为包括：

- 使用性语言或性图像
- 骚扰、侮辱或贬低性评论
- 个人或政治攻击
- 公开或私下骚扰
- 在无明确同意的情况下发布他人的私人信息，如地址或电子邮件
- 其他可能被合理认为在职业环境中不适当的行为

## 执行原则

项目维护者负责为所有参与者建立和执行可接受行为的标准。维护者有权利和责任删除、编辑或拒绝不符合本准则的评论、提交、代码、wiki 编辑、问题和其他贡献。

## 报告程序

如果你目睹或遭到骚扰、虐待或无法接受的行为，请通过以下方式与项目维护者联系：

- 发送邮件至：contact@example.com
- 通过 [私密问题](https://github.com/yourusername/Love-Cube/security/advisories) 报告
- 在 GitHub 中报告用户

所有举报都将进行审查和调查，并将导致被认为必要和适当的应对措施。

## 隐私

项目维护者致力于为举报人保护隐私。

## 后果

维护者认为违反此行为准则的行为将导致以下后果，取决于严重程度：

1. **口头警告** - 首次轻微违反
2. **在社区中暂时禁言** - 对于中等或重复违反
3. **永久禁止** - 对于严重或重复的不可接受行为
